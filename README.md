# Debtemp

## Overview

This generates a basic DEBIAN package from Github-name/repo-name.

## Requirement

- Linux/Debian

- dpkg-deb

Debian/Ubuntu should have it. If not install it using apt.

```sh
$ sudo apt install dpkg
```

## Usage

```sh
$ debtemp shinokada/teffects
```
